# Zomato Data Analysis (Power BI)

## Project Overview
This project focuses on analyzing Zomato restaurant data to uncover insights about customer preferences, restaurant performance, and pricing trends. Using Power BI, interactive dashboards were created to visualize key business metrics and support data-driven decision-making.

## Objectives
- Analyze restaurant ratings and customer votes
- Identify top-performing restaurants and cuisines
- Understand pricing trends across different locations
- Evaluate the impact of online delivery and table booking
- Discover patterns in customer preferences

## 🛠️ Tools & Technologies Used
- Power BI (Data Visualization & Dashboarding)
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

## Key Insights
- Restaurants with higher ratings tend to receive more customer votes
- Certain cuisines dominate in popularity across multiple cities
- Pricing varies significantly based on location and restaurant type
- Online delivery availability impacts customer engagement
- Metro cities have higher concentration of highly-rated restaurants
